GreenRater's Production Process

The Hypothesis

How can we use digital tools to create a geospatial-based criteria to evaluate Fulton County's greenspaces?

Ideation

Developing a Concept (November)

GreenRater is a continuation from a 2023 project titled ParkSpot, a website allowed users to view a database of local greenspaces in their area. Building on this strong foundation, the project's goal shifted from simple data presentation to utilizing data insights to build an impactful multi-functional digital tool.

PocketPark Prototype Presentation

Method Selection

Selecting the Best Method (December)

Based on guidance from mentors and professionals in urban design space, content analysis and a digital prototype were the two methods chosen to direct GreenRater's development.

Source Collection

Preparing for Content Analysis

In the Source Collection stage, a total of 60 diverse sources were selected for further analysis. These greenspace-evaluation texts ranged from governmental information pamphlets to case studies using certain geospatial tools to broader academic pieces about the evaluation process.

15 Sources Final 60 Sources

Content Analysis

Analyzing 60 Sources

Using SQL, all selected sources were categorized by findings, geospatial tools, topic, and criteria category. This process allowed for more accurate pattern recognition of successful greenspace evaluation metrics among the texts.

Community Survey

Dissecting Alpharetta's Greenspace Sentiment

To incorporate community input into this project iteration, 45+ responses were collected from local Fulton County residents. These responses enabled our team to analyze local sentiment on the current state of Fulton County’s greenspaces and community needs. Questions evaluated the frequency and quality of greenspace visits as well as inquired for any current issues.

Greenspace Evaluation Criteria

Final Criteria

Using and Evaluating Results

The final GreenRater scoring model is based on a five-category weighted system, combining:

  • 65% community input
  • 35% research-based findings

    • This approach ensures the scoring system is both evidence-based and reflective of real user experiences.
